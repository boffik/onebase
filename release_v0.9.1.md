## Что нового

`v0.9.1` — патч-релиз вокруг развёртывания, offline-обновлений и стабильности пользовательских сценариев. В него вошли команды для обслуживания инсталляций без доступа к интернету, readiness-проба для мониторинга, REST API для вложений и несколько исправлений в DSL, UI и импорте архивов.

## Изменения

- **Offline-обновление платформы** — `onebase update --from <archive>` умеет обновляться из локального архива и откатываться при ошибке.
- **Hot reload database-конфигураций** — `--watch` подхватывает изменения конфигурации без ручного рестарта, включая установку сервиса через `service install --watch`.
- **Readiness и бэкапы** — добавлен `/healthz` для проверок готовности и CLI-сценарий SQLite-бэкапа.
- **REST API v2 для вложений** — добавлены endpoints загрузки, скачивания и удаления вложений с учётом row-level access.
- **Печатные формы** — поддержан кастомный размер страницы в миллиметрах и добавлен пример многостраничной регламентной формы.
- **Навигация** — конфигурация может скрывать глобальную «Главную», оставляя навигацию только по разделам.
- **Документация развёртывания** — добавлено руководство по Windows и offline-сценариям.

## Исправления

- Защита импорта конфигураций и бэкапов от zip-slip (CWE-22/23).
- Маскировка API-ключа ИИ в конфигураторе и корректное копирование конфигурации текстом.
- Направление раскрытия dropdown «Печать» на managed-формах документов.
- SSE под лаунчером: `statusRecorder` теперь пробрасывает `Flush`.
- DSL: `+` над двумя строками всегда выполняет конкатенацию; сохранены строковые преобразования и lookup ссылок.
- Уведомления/toasts доставляются стабильнее через realtime hub.
- Добавлены недостающие i18n-ключи для конфигуратора, архива и загрузки вложений.
- CI и release workflow закреплены на Go `1.26.5`.

## Установка

### Windows
1. Скачайте `onebase-windows-amd64.zip`
2. Распакуйте в любую папку, например `C:\onebase\`
3. Запустите `onebase-gui.exe`

### Linux
```bash
tar xzf onebase-linux-amd64.tar.gz
./onebase-linux-amd64/onebase start
```

### macOS
```bash
tar xzf onebase-darwin-amd64.tar.gz
./onebase-darwin-amd64/onebase start
```

### Offline-обновление
```bash
onebase update --from D:\flash\onebase-v0.9.1.zip --id <база>
```
