## 🎯 Главное

Релиз v0.5.0 — переходный: OneBase уходит от обязательного PostgreSQL к **SQLite «из коробки»**, получает собственную систему валидации конфигураций и инструменты для AI-ассистентов. Дашборды и формы стали удобнее, а публичная демо-база — проще в развёртывании.

## ✨ Новое

- **SQLite по умолчанию** — первый запуск без PostgreSQL: `onebase start` создаёт локальную базу автоматически. Полная поддержка DSL, запросов и бэкапов.
- **`onebase check`** — валидация конфигурации без запуска сервера: синтаксис YAML, запросы в модулях, кросс-ссылки между объектами, поля журналов и печатных форм. Точный адрес ошибки + exit code для CI.
- **`onebase ai-guide`** — генерация справочника для ИИ: функции DSL, схема метаданных, частые ошибки запросов. Флаг `--claude` пишет `CLAUDE.md` автоматически.
- **Самохостинг Monaco** (#17) — редактор кода встроен в бинарник, конфигуратор и формы работают полностью офлайн, без CDN.
- **Периодичность оборотов** — виртуальная таблица `Обороты` теперь поддерживает периодичность (день/месяц/квартал/год), алиас `period` для колонки.
- **«Ввести на основании»** — меню на форме объекта для создания связанных документов по аналогии с 1С.
- **Drag-конструктор дашбордов** — режимы раскладки `auto/rows`, перетаскивание виджетов, живой предпросмотр теми же ECharts.
- **Файловое поле** в управляемых формах — выбор файла через пикер или вручную, содержимое доступно в DSL.
- **Сворачиваемые группы меню** — навигация подсистем с группировкой и правами на обработки.
- **`demo-reset` + `--demo-*`** — команда сброса демо-базы из `.obz` + флаги для демо-режима при запуске.
- **Бухгалтерская конфигурация** — полноценный пример с планом счетов, проводками и субконто.

### Валидация (`onebase check`)

Трёхуровневая система проверок:
- **Статическая** — YAML-схема, типы полей, ссылки на несуществующие объекты
- **Кросс-ссылочная** —孤儿-ссылки между документами, справочниками, регистрами
- **Исполняемая** — компиляция запросов в in-memory SQLite, выявление ошибок SQL до запуска

### AI-инструменты

`onebase ai-guide` генерирует актуальный справочник:
- Все встроенные функции DSL с сигнатурами
- Схема объектов метаданных (документы, справочники, регистры)
- Чек-лист частых ошибок в запросах
- `--output AGENTS.md --claude` — готовый файл для Claude Code

## 📦 Установка

### Windows
1. Скачайте `onebase-windows-amd64.zip`
2. Распакуйте в любую папку
3. Запустите `onebase-gui.exe` или `onebase start`

### Linux
```bash
tar xzf onebase-linux-amd64.tar.gz
./onebase serve
```

### macOS
```bash
tar xzf onebase-darwin-amd64.tar.gz
./onebase serve
```

### Требования
- PostgreSQL 14+ — только для многопользовательского режима
- Для SQLite-режима дополнительных зависимостей нет
