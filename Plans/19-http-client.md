# Этап 19 — HTTP-клиент в DSL

**Статус:** ⬜ Не начато

## Контекст

В DSL нет возможности обращаться к внешним API. Это блокер для интеграций: загрузка курсов валют, работа с маркетплейсами, Telegram-уведомления, банк-клиент. Регламентные задания есть, но без HTTP-клиента они бесполезны для интеграционных сценариев.

## Синтаксис

```
// GET-запрос
Соединение = Новый HTTPСоединение("api.cbr.ru");
Запрос = Новый HTTPЗапрос("/v1/rates");
Ответ = Соединение.Получить(Запрос);

Если Ответ.КодСостояния = 200 Тогда
    данные = ПрочитатьJSON(Ответ.ПолучитьТелоКакСтроку());
КонецЕсли;

// POST-запрос с заголовками и телом
Соединение = Новый HTTPСоединение("api.example.com", 443, Истина); // HTTPS
Запрос = Новый HTTPЗапрос("/orders");
Запрос.УстановитьЗаголовок("Content-Type", "application/json");
Запрос.УстановитьЗаголовок("Authorization", "Bearer токен");
Запрос.УстановитьТелоИзСтроки("{""status"":""new""}");
Ответ = Соединение.ОтправитьДля(Запрос, "POST");

// Коротко: GET по URL одной строкой
Ответ = HTTPПолучить("https://api.example.com/data");
текст = Ответ.ПолучитьТелоКакСтроку();
```

## Объекты DSL

### HTTPСоединение
- `Новый HTTPСоединение(хост [, порт [, HTTPS [, таймаутСек]]])`
- `.Получить(запрос)` → HTTPОтвет
- `.ОтправитьДля(запрос, метод)` → HTTPОтвет

### HTTPЗапрос
- `Новый HTTPЗапрос(ресурс)`
- `.УстановитьЗаголовок(имя, значение)`
- `.УстановитьТелоИзСтроки(текст)`

### HTTPОтвет
- `.КодСостояния` → число
- `.ПолучитьТелоКакСтроку()` → строка
- `.ПолучитьЗаголовок(имя)` → строка

### Shorthand-функции
- `HTTPПолучить(url)` → HTTPОтвет
- `HTTPОтправить(url, тело [, заголовки])` → HTTPОтвет

## Безопасность

- Whitelist разрешённых хостов в `app.yaml` (опционально): `http_allowlist: [api.cbr.ru]`
- По умолчанию — без ограничений (платформа для разработчиков)
- Таймаут по умолчанию: 30 секунд
